Osiris

Osiris is a small Open Source CRM / Sales Activity Management system. SMS notifications are part of its backbone. You can monitor your prospects while staying alert to upcoming tasks. It is small, quick, portable and easy to use.

Download Now!

It's easy to test drive! Download the project, untar then run the following command :

mvn package jetty:run

Then browse to: 

http://localhost:3001
