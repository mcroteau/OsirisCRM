Polygon

Polygon is a small Open Source CRM / Sales Activity Management system. SMS notifications are in its backbone. You can monitor your prospects while staying alert to upcoming tasks. It is small, quick, portable and easy to use.

Download Now!

It's easy to test drive! All you need is Java. Download the project, untar then run the following command :

./mvnw package jetty:run

Then browse to: 

http://localhost:3001
